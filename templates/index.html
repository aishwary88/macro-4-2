Phone Camera Setup Guide

1

Install IP Camera App

Android: IP Webcam Pro, DailyRoads Voyager

iPhone: Iriun WebCam, Reincubate Codeshot

2

Start Camera Server

Open app → Grant permissions → Tap "Start Server"

Note the IP address shown (e.g., 192.168.1.100:8080)

3

Find Stream URL

IP Webcam Pro: http://192.168.x.x:8080/video

DailyRoads: http://192.168.x.x:8080/mjpegfeed

Test URL in browser first!

4

Connect to Dashboard

Paste URL in "Camera Source" field above

Click "Start Camera" → Live video appears!

Troubleshooting

  • Ensure phone and laptop on same WiFi
  • Check app is actually running on phone
  • Try different endpoint paths (/video, /mjpegfeed)
  • Disable phone firewall temporarily
